DemandResponseLoadControlCluster¶

class st.zigbee.zcl.clusters.DemandResponseLoadControlCluster: ZigbeeCluster¶

This cluster provides an interface to the functionality of Smart Energy Demand Response and Load Control. Devices targeted by this cluster include thermostats and devices that support load control.

ID: number¶

0x0701 the ID of this cluster

NAME: str¶

“Demand Response and Load Control” the name of this cluster
