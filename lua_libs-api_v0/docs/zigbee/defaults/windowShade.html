static default_current_lift_percentage_handler(ZigbeeDriver, ZigbeeDevice, value, zb_rx)¶

Default handler for current lift percentage attribute on the window covering cluster

This converts the Uint8 value to Open, Close or partially open event

Parameters

  • ZigbeeDriver (Driver) – The current driver running containing necessary context for execution

  • ZigbeeDevice (ZigbeeDevice) – The device this message was received from containing identifying information

  • value (Uint8) – the value of the current lift percentage of the window covering cluster

  • zb_rx (ZigbeeMessageRx) – the full message this report came in
