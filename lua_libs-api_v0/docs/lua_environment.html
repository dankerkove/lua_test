Lua Environment¶

SmartThings Lua scripts execute in an environment similar to the stock Lua environment.

The execution environment available to scripts

environment.report_error(...)¶

Report a script error

This currently reports to Sumo, but is expected to be surfaced to users via dev-workspace

Parameters

vararg (any) – arbitrary values to report

environment.require(name, version)¶

Load a lua file

Parameters

  • name (str) – file to load

  • version (unused) – Unused

environment.sleep(seconds)¶

Sleep for an arbitrary amount of time

Parameters

seconds (number) – Sleep duration
